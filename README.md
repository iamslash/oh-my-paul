# oh-my-paul

## Status

**문서 작성 중** — 아직 구현 전

## Quick Start

```bash
# Claude Code에 설치
claude /plugin install https://github.com/iamslash/oh-my-paul

# 사용
/commit        # git commit 자동화
/review        # 코드 리뷰
```
