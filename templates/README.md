# oh-my-paul Templates

프로젝트에 복사하여 사용할 수 있는 규칙 템플릿.

## 사용법

```bash
# 전체 복사
mkdir -p .paul/rules
cp /path/to/oh-my-paul/templates/rules/*.md .paul/rules/

# 개별 복사
cp /path/to/oh-my-paul/templates/rules/coding-style.md .paul/rules/
```

## 규칙 목록

| 파일 | 설명 |
|---|---|
| coding-style.md | TypeScript 코딩 스타일 |
| git-workflow.md | Git 커밋/브랜치 규칙 |
| testing.md | 테스트 작성 규칙 |
| security.md | 보안 코딩 규칙 |
